# SFMS (Safety Facility Management System) Admin Demo

시설물 안전 관리 및 고객 소통을 위한 관리자 대시보드 데모 사이트입니다.  
현재 프론트엔드 UI/UX 프로토타입으로 구현되어 있으며, 추후 MySQL DB 및 카카오톡 API와 연동될 예정입니다.

## 🛠 주요 기능

### 1. 관리자 로그인 (`index.html`)
* 보안을 위한 관리자 인증 페이지입니다.
* 지정된 관리자 계정으로만 대시보드에 접근할 수 있습니다.

### 2. 가입정보 관리 (`main.html` - Tab 1)
* 시설물 관리자 및 담당자의 가입 정보를 테이블 형태로 조회합니다.
* **표시 항목:** 번호, 기관 구분, 기관명, 담당자 직책, 지역, 가입일시
* 현재는 Mock Data(샘플 데이터)를 사용하여 화면을 렌더링합니다.

### 3. 메시지 발송 시스템 (`main.html` - Tab 2)
* 특정 조건의 고객을 필터링하여 단체 메시지(알림톡)를 발송하는 기능입니다.
* **필터링 옵션:**
    * 지역 (송파, 잠실, 서초 등)
    * 기관 구분 (어린이집, 구청, 시설물)
    * 직책 (안전담당자, 원장)
* **실시간 대상 조회:** 필터 조건을 변경하면, 발송 대상 리스트와 인원수가 실시간으로 업데이트됩니다.
* **발송 시뮬레이션:** 제목과 본문을 입력하고 보내기 버튼을 누르면 발송 성공 알림창이 뜹니다.
